Create and activate virtual environment in a location of your choosing:
	python3 -m venv mkdocs.venv
	source mkdocs.venv/bin/activate
Clone mkdocs-material in a location of your choosing:
	git clone https://github.com/squidfunk/mkdocs-material.git
Install mkdocs-material and plugins
	pip install -e mkdocs-material
	pip install mkdocs-awesome-pages-plugin

At least for the moment, there's a bug getting things to rebuild when a file
updates.  The fix is:
   pip install "click<8.3"
